# Nexum Agent Skills

Project-local agent skills for Nexum Network work.

Each skill lives in its own directory and contains a `SKILL.md` file with:

- when to use it,
- expected inputs,
- workflow,
- output,
- safety notes.

These are not automatically installed global Codex skills. They are source-controlled project instructions that can be copied into an agent runtime or used as task briefs.
